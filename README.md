# 🌐 Meus Sites Favoritos

Projeto desenvolvido em HTML para a atividade prática da disciplina de Desenvolvimento Web.

## 📖 Sobre o Projeto

A proposta da atividade foi criar uma página web contendo:

- 5 sites favoritos;
- Imagens dos sites;
- Links de acesso;
- Estrutura HTML organizada;
- Publicação do projeto no GitHub.

## 🚀 Sites Utilizados

- YouTube
- Google
- Instagram
- ChatGPT
- Gemini
## 🛠️ Tecnologias Utilizadas

- HTML5
- VS Code
- GitHub
